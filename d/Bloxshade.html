Instructions

You can't just run the exe and expect shaders. There is some required manual setup.

Pre-requisite: Install & Setup Nvidia App

  1. Download and install Nvidia App
  2. Launch Nvidia App and go to settings
  3. Enable Nvidia overlay and game filters and photo mode
  4. Make sure game is closed

Using Bloxshade

  1. Launch Bloxshade.exe as downloaded above
  2. Click install and wait for it to list available shaders
  3. Install as many shaders as you find interesting
  4. Click finish
  5. Join a game
  6. Press alt + f3
  7. In the newly opened Nvidia menu, click Game filter
  8. Choose any filter you want in the list. The setup should've given you options to add to the list. You should now be able to observe changes to your game's visuals.

FAQ


What is this?

This takes advantage of Nvidia Ansel to change your game filters. Think of it it as a hub of community made presets render style presets.

Will it cause lag?

Reshading can be intense. Even a high budget PC can lag if you start to layer too many filters.

The game updated and it broke?

Try reinstalling Bloxshade.

How do I launch Bloxshade after installing?

Bloxshade is more of a preset downloader. Bloxshade doesn't have an interactive UI, rather, you'll be using the Nvidia overlay to control the game's appearance. When you join a game, press alt + f3 to launch the Nvidia overlay.
